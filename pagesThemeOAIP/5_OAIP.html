Задание №1

Нужно написать библиотеку, с помощью которой можно будет вычислить дискриминант.

Теория

Здесь нужно будет создать 2 проекта, в одном из которых будет прописана dll-библиотека. Для начала займемся библиотекой.

Решение

Далее в основном проекте, импортировав библиотеку, прописываем переменные, получаем ввод от пользователя и вызываем класс для решения дискриминанта.
